== Description ==
This plugin is made for learning purpose. It has functionality like Creating static or fetch dynamic post type. 
This plugin has block in the backend. It does not has any functionality in the frontend.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory. This plugin is not available in WordPress plugin ORG. DIR
1. Activate the plugin through the 'Plugins' screen in WordPress
